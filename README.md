# Atividade Prática - Criação e Edição Multimídia 

Este repositório contém o código-fonte desenvolvido como parte de uma atividade prática para o curso **Técnico em Desenvolvimento Web e Mobile**, especificamente para a disciplina de **Criação e Edição Multimídia**.

O objetivo deste projeto é aplicar os conceitos de estruturação de pastas, criação de páginas HTML estilizadas com Tailwind CSS e a integração de diferentes formatos de mídia (imagens, vídeos e áudios), tanto de diretórios locais quanto incorporados de plataformas externas.

## Funcionalidades e Páginas do Projeto

O projeto foi dividido em várias páginas HTML, cada uma com um propósito específico exigido na atividade:

*   **Página Principal (`index.html`)**: Página base do projeto que serve como um menu de navegação para as outras seções. O rodapé foi personalizado com os devidos direitos autorais.
*   **Galeria de Imagens Local (`gllocal.html`)**: Uma galeria interativa de imagens carregadas a partir de uma pasta local do projeto. Inclui visualização ampliada (modal) e a tag de identificação do autor.
*   **Vídeo do YouTube Incorporado (`vyincorpo.html`)**: Uma página dedicada a exibir um vídeo incorporado diretamente do YouTube, mantendo a responsividade adequada.
*   **Áudio Local (`alocal.html`)**: Demonstração da tag de áudio do HTML5, reproduzindo um arquivo salvo na pasta local do projeto.
*   *(Nota: Todas as subpáginas possuem marcações no código e subtítulos (`<h2>`) identificando o desenvolvedor responsável).*

## Estrutura de Pastas

Para manter a organização exigida, o projeto utiliza a seguinte estrutura de diretórios:
```text
/
├── audios/         # Arquivos de áudio (.mp3, .m4a, etc.)
├── imagens/        # Arquivos de imagem (.jpg, .png, etc.)
├── paginas/        # Subpáginas do projeto (.html)
├── videos/         # Arquivos de vídeo locais (.mp4, etc.)
└── index.html      # Página principal
